# 결론
현재 기후동행카드는 서울에서만 가능하고 지하철역 상당수와 서울을 자주 오가는 경기도와 인천 지역에서 사용이 불가능하다.  
서울외의 수도권에서도 대중교통 수요가 많기 때문에 서울 외 수도권에서도 기후동행카드를 지원해줄 필요가 있다.
